Devtools - Jump to Debugger

Oct 15th, 2013 | Comments

Here’s a quick trick to jump into a breakpoint without having to hunt through the sources for the function. Assume you have something like this

It would be nice if we could just jump to the function with a breakpoint toggled. Here’s one way to do that: Type in > (function() { debugger; theFunctionToStepThrough(); })() Now we get to this screen And if you hit F11 or click the step into button we get to this: Hope it comes in handy

It seems that you don’t have to even wrap it in an IIFE, instead all you need to do is:

> debugger; test();
